# Contributing to GravityJS

Thanks for your interest in contributing!

## Development setup

1. Clone the repository
2. Install dependencies with `npm install`
3. Build with `npm run build`

## Pull requests

- Keep changes focused and small where possible
- Include tests or demos for new behaviour where it makes sense
- Update documentation when you change public APIs or attributes

## Reporting issues

Please include:

- A clear description of the problem
- Steps or a minimal reproduction (CodeSandbox, repo link, or HTML snippet)
- Your browser(s), framework (if any), and GravityJS version
